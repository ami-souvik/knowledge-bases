# Agent Tree Overview

**Agent Tree** is an evolving platform designed to automate businesses leveraging agentic AI. 

## History & Context
The project initially started and successfully deployed a platform exactly similar to `chatbase.co`, allowing users to quickly assemble and train chatbots for specific contexts. 

## Current Paradigm Shift
The platform is expanding its core capabilities. Rather than being confined to chatbots, Agent Tree will serve as a multi-agent orchestration platform where users can construct workflows. Autonomous agents will be able to interface directly with external platforms and tools, automating tedious operations, generating content, managing data, and more.

## High-Level Capabilities (Planned)
- Custom AI chatbots trained on internal data (existing)
- Multi-agent workflow automation
- Direct external integration to existing business apps
- Scalable, Next.js + serverless/edge-friendly architecture
