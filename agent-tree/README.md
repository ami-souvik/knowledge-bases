# Agent Tree

**Currently Live at:** [agent-tree.vercel.app](https://agent-tree.vercel.app/)

Agent Tree is a platform aimed at **automating businesses leveraging agentic AI**. 

## Vision & Origins
Agent Tree originally began as an AI platform similar to `chatbase.co`, allowing users to build custom AI chatbots trained on their own data. We are now expanding this vision to build a holistic automation system where autonomous agents can execute business workflows, integrate with existing platforms, and intelligently manage tasks beyond conversational chat.

## Architecture & Focus
We are building robust multi-agent systems and leveraging modern web frameworks (Next.js, TypeScript) to create an interface that seamlessly integrates AI workflows into daily business operations.
